# hotels_parser

Программа парсер данных по сайту с отелями, через Selenium


Запуск:
1) Скачать и установить python3
2) Склонировать проект
3) Открыть терминал (linux)
4) Перейти в папку проекта
5) Создать виртуальное окружение командой python3 -m venv venv
6) Активировать виртуальное окружение командой source /venv/bin/activate
7) Установить зависимости командой pip install -r /src/requirements.txt
8) Перейти в каталог src
9) Запустить скрипт командой python3 hotel_parser.py
